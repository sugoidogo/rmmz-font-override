# Sugoi Font Override
Adds Font Overrides to Options
# Usage
Download the plugin from releases and put it in the plugins folder of your project.
Create an `alt_fonts` folder in your project folder and place your extra fonts inside.
Playtest the game, the plugin will generate an `alt_fonts/index.json` file for the web/mobile distro.
Your game will now have an extra option to allow the player to override the default font.
